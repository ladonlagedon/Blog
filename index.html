关于找房子要问的问题

Jun 26, 2019

因为太杂乱所以总结出一些要点：

问清楚价格，是否为最终的价格，还有无其他费用，还要了解各种杂七杂八的费用怎么交。

押金交多少，第二年交不交，中途退房是否能退押金，退多少，逾期没交房租被赶走，退押金不。

实地看房，感受环境，噪音，隔音之类的，东西烂了怎么赔偿，是否影响押金退费。

要求房主提供有效证明，包括但不限于身份证，房产证，出租房屋的证明。

房租从多久起开始清算，什么时候交给房主，看好房后，了解定金去向，是否包含在房租内。

签合同的时候注意观察是否还绑定了其他的合同，是否有“无论通没通过中介都要交中介费”的条款。

（原答案）实习加紧看房子并总结的一些问题，如下：

洗衣机是否公用？是否独立卫生间？（个人需求）

个人房间有锁不？晒衣服在哪里？

洗澡间的和厕所在一起吗？厕所和洗漱台在一起吗？（个人需求）

房租一共多少？包含其他费用没有？

中介费？服务费？手续费？介绍费？看房费？咨询费？（问清楚杂七杂八的费用）

水费？气费？电费？网费？物业费？维修费？保洁费？（这一块主要问怎么算的，平摊还是单独算）

宽带是否公用？规格（20兆 50兆 100兆）是多少？不满意能自己换不？换了还交公用宽带费不？

付款模式？押一付一？押一付三？

// JavaScript 中的隐式转换，只存在于非 '+' 的运算中，因为 '+' 既是连字符也是加号。
// 只要 undefined 参与运算 结果都是 NaN
// true false null 都会隐式转换 纯数字字符串会隐式转换(加法除外)

console.log(3 + 5); // 8

console.log(3 / '5'); // 0.6

console.log('3' + '5'); // 35
console.log('3' - '5'); // -2
console.log('3' * '5'); // 15
console.log('3' / '5');

console.log(3 * true); // 3
console.log(3 / false); // Infinity
console.log(3 / null); // Infinity
